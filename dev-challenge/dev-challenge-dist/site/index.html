Assumptions

  • I assumed that sorting should be in ascending order by last changed bid
  • I assumed that we need to show sparkline in last column of every row
  • I assumed that if we get the data with the same name, we need to merge it

Approach

  • Created utils.js module using CommonJS, which contains all the required functions
  • Added timestamp attribute to manage last 30 seconds data to be shown as a sparkline
  • To avoid multiple entries in sparkline array related to any particular name, we are updating it each time with filtered data of last 30 seconds
  • Added meta viewport element,So that web page can adjust its width according to viewport
  • Added required styling to make page responsive with horizontal scroll for table
  • Added jasmine for unit testing
  • By default jasmine don't show report of successful unit tests, So I have modified the report to show success logs with details
  • You can find the test files in "spec/test_files" location

Steps to run the assignment

  • npm install
  • npm start

Steps to run the unit tests

  • npm test
