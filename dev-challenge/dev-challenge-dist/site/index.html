Assumptions

  • I assumed that sorting should be in ascending order by last changed bid
  • I assumed that we need to show sparkline in last column of every row

Approach

  • Created utils.js module using CommonJS, which contains all the required functions
  • To avoid re-rendering of table I am finding the index where new data has to be inserted and adding the row at that particular index of table
  • Added timestamp attribute in each received object, to manage last 30 seconds data to be shown as a sparkline
  • Added jasmine for unit testing
  • By default jasmine don't show report of successful unit tests, So I have modified the report to show success logs with details
  • You can find the test files in "spec/test_files" location

Steps to run the assignment

  • npm install
  • npm start

Steps to run the unit tests

  • npm test
